# zoned.

A Next.js App Router prototype for the zoned. study focus app.

## Features
- Parent and child role switcher
- Study session monitoring and idle alerts
- App lock controls and automatic blocking options
- Task assignment and completion rewards
- Leaderboard, friend chat, and mystery box rewards
- Supabase-ready helpers for auth and session management

## Setup

1. Create `.env.local` with your Supabase environment variables.
2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 to view the app.
